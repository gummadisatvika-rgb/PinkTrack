# PinkTrack
My website
 

PinkTrack is a privacy-first menstrual cycle tracker designed with a soft pastel aesthetic and empowering features. Built using HTML, CSS, and JavaScript, it helps users log their period, track symptoms, and visualize their cycle with countdowns and calendar views.



 Features

  Period start/end tracking
  Symptom and mood logging
  Calendar view of cycle days
  Countdown to next period and ovulation
  Editable notes per entry
  Reminder toggle
  Rotating health tips





Frontend-HTML, CSS, JavaScript
Backend-Node.js + Express (free and lightweight)
Storage- JSON file (local for now)
Deployment-AWS EC2 (Ubuntu + NGINX)
